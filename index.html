Welcome to ZHAW's Neural Art Project.

This page shows results given by neural_style, a fast generative image style transferer implemented with mxnet.

There are lots of high resolution images on this page and it may take 1 or 2 minutes loading, please be patient.

Content Images

These are the images I selected to test and visualize models. These images vary a lot so we can know the robustness of the models.

Styles

These are the style models I have trained, more will come. For better page layout, images are resized to 512x512. Please notice that scale matters. If the result isn't good, try resize the image. If you want to use one style model, clone this repo, download the model and put it in the "models/" dir. For more usage guidance, see this.
